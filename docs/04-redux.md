# 4th Practical Class: Redux

## Install

```bash
git fetch --all
git reset --hard cngroup/practical-04
cd frontend
npm install
mkdir tmp
touch tmp/restart.txt
cd ../backend
npm run build:watch
```

Change your API URL in `frontend/src/api.js`
