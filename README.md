## How to run
### frontend dev
1. go to the frontend folder
```
cd frontend
```
2. install dependencies
```
yarn
```
3. start the project
```
yarn start
```
4. Good job! Your frontend is running default at: `localhost:3000`


### backend dev
1. go to the backed folder
```
cd backend
```
2. install dependencies
```
yarn
```
3. start the project
```
yarn dev
```
4. Good job! Your backend is running default at: `localhost:3030`
